# Glome Headless

This runs the glome in headless mode.  The model is in `GlomeModel.java` and reads a `pixels2.json` file from where you execute the file.

Playlist is defined in `testing.lxp` and 

## Build
```sh
ant
```

## Run
```sh
./GlomeHeadless testing.lxp
```
